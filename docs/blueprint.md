# **App Name**: FunToKnow Platform

## Core Features:

- Project Showcase: Display game projects with details (title, description, media, links), sourced from Firestore.
- Blog Publication: Publish blog posts dynamically (/blog/[slug]) with markdown support, also managed in Firestore.
- Contact Form: A contact form that saves user messages to Firestore for review.
- User Authentication: Enable user registration, login, and profile management using Firebase Auth.
- Admin Panel: Admin interface for CRUD operations on projects, blog posts, and users, protected with role-based access (Firebase Auth).
- AI-Powered Suggestion Tool: AI tool to analyze user's gaming habits from Firestore 'user' collection to provide a tailored list of projects which can improve the experience.

## Style Guidelines:

- Primary color: Deep violet (#673AB7), symbolizing creativity and digital artistry.
- Background color: Light gray (#F5F5F5), providing a clean and modern canvas for the content.
- Accent color: Electric purple (#8A2BE2) for interactive elements and call-to-actions, enhancing user engagement.
- Electric blue (#7DF9FF) to highlight interactive elements.
- Orange (#FFA500) to indicate warnings or important actions.
- Normal blue (#0000FF) to highlight interactive elements.
- Headline font: 'Poppins', a geometric sans-serif for a contemporary, fashionable look.
- Body font: 'PT Sans', a humanist sans-serif for readability and a touch of warmth.
- FontAwesome icons for a consistent and visually appealing UI, representing different game genres and actions.
- Bootstrap 5 grid layout to ensure responsiveness and a clean content structure across devices.
- Subtle transitions and animations to enhance user experience, such as loading indicators and button hover effects.
